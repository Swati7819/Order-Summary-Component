# Order Summary Component

## Overview

This project is a simple order summary card designed to showcase a fictional service's annual subscription plan. It provides users with a clear summary of their order and prompts them to proceed to payment.

## Technologies

- HTML
- CSS

## Features

- Responsive design ensures usability across various devices.
- Clean and modern UI enhances user experience.
- Interactive "Proceed to Payment" button allows users to take action directly from the card.
